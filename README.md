A simple terminal-based C++ application that helps users track daily habits, set goals, and visualize progress over time.

## Features

- Add, remove, and track habits
- Daily and weekly progress reports
- Goal completion tracking
- Data persistence using CSV files

## Installation

```bash
git clone https://github.com/codcreater1/habit-tracker.git
cd habit-tracker/src
g++ main.cpp habit.cpp -o habit-tracker
./habit-tracker
```

## Usage

- Follow on-screen instructions to add habits and log progress
- Check weekly statistics to track your improvement
